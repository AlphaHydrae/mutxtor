    public get delta: integer: number source

    The positive or negative difference of the total length of the text document after the mutation is applied.

    • If the mutation adds 3 characters, the delta is 3.
    • If the mutation adds 5 characters and removes 7, the delta is -2.

    Return:

    integer

    The number of characters added to or removed from the document by this mutation.

    public element: TextElement source

    The text element being mutated.

    public get end: integer | undefined: * source

    The end index at which this mutation takes place in the text document.

    If the mutation affects an existing range of text, this must be the end index (exclusive) of that range. If the mutation only adds new text, this must be undefined.

    Return:

    integer | undefined

    The end index (exclusive) of the existing text range affected by this mutation, or undefined if the mutation only adds new text.

    public get start: integer: * source

    The start index at which this mutation takes place in the text document.

    If the mutation affects an existing range of text, this must be the start index (inclusive) of that range. If the mutation only adds new text, this must be the index at which that text will be inserted.

    Return:

    integer

    The index at which this mutation takes place in the text document.
