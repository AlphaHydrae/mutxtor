Class ParserBuilder

A chainable builder to add a text parser to a TextDocument.

TextDocument parsers are generator functions that must return TextElement objects, usually a subclass. This builder simplifies making such a function.

example

// Adds a parser that will prepend a horizontal rule before all titles in a Markdown document (except the main title). document.buildParser('MarkdownTitle').regexp(/^(#+)\s*(.+)$/gm).mutate(function(data) { const titleLevel = data.match[1].length; if (titleLevel >= 2) { this.prepend('---\n'); } }).add();
